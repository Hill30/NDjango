
NDjango Requirements:
 *Visual Studio 2008 or(and) Visual Studio 2010

NOTES:
1) Remove all previous of NDjango using Programs and Features menu before you run this setup!
2) To install NDjango project templates for VS2008 you must have ASP.NET MVC 2 and ASP.NET MVC 2 - Visual Studio 2008 Tools installed on your computer.
3) To use NDjango Editor in your Visual Studio 2010 you must have FSharp Power Pack 1.9.9.9 installed.

NDjango files to be installed:

1. NDjango.Core.dll(.Net 3.5,.Net 4.0)
2. NDjangoFilters.NDjangoExtension.dll(.Net 3.5,.Net 4.0)
3. NDjango.Designer.vsix
4. ASPMVCIntegration.dll(.Net 3.5,.Net 4.0)
5. StructureMap.dll
6. NDjango templates
7.ASP MVC 3 Preview Add View Code Templates


This setup is no longer includes BistroIntegration and CastleIntegration assemblies.
